Let's start with some general advice about the game. Firstly, go to the Ubisoft website and download the V1.03 patch for the game. You will be able to tell when you have installed this latest patch because each time you load the game the version number 1.03 should appear on the bottom left of the screen
     
 

There are a couple of "timed" puzzles in Myst IV - that means you need to do certain things within a time limit. The original release had very short time limits on some of these puzzles so without the V1.03 patch, you will find them very difficult (also my solution to what is widely claimed to be the most difficult puzzle, will not work)
     
 

The menu on the left will take you to the walkthrough for each of the ages in Myst IV. I've tried to lay them out in a way that means you can get a bit of guidance on what to do before the solution is spelled out and these pages will help to explain the reasoning behind alot of the puzzles or events.

However, if you are just seeking help with one particular puzzle, you can link to advice on individual puzzles here.
